# 안압케어 (AnapCare) — 녹내장 통합관리 앱

CVT200 안압계를 중심으로 **안압 · 점안 · 전자문진 · 웨어러블 건강데이터**를 하나의 환자 앱에 통합하고,
임상적으로 의미 있는 데이터를 **의료진 웹**으로 전달하는 녹내장 통합관리 솔루션의 프로토타입입니다.

> C&V Tech · CVT200 Companion · **프로토타입 / 데모용 (샘플 데이터)**

---

## 주요 기능

### 환자 앱
- **안압 (IOP)** — CVT200 실시간 다회 측정. 실시간 값은 측정값 그대로 표시하고, 특정 일자는 그날 측정값의 **평균**으로 계산하며 **최소–최대(일중 변동)** 범위를 함께 표시합니다. 기간 추세는 2주 / 1개월 / 3개월 / 6개월 / 1년 / 누적으로 조회할 수 있습니다(기본 1개월).
- **점안** — 성분명별 점안제 라이브러리(제조사 · 일회용/다회용 구분) 선택 또는 수기 입력, 순응도 추적. 대우제약 제품이 상단에 노출됩니다.
- **전자 문진 (12항목)** — 입력 빈도(초기/주/월/분기)로 구조화. 환자는 옵션 선택만 하며, 위험도·맞춤안내는 노출하지 않습니다.
- **건강 (웨어러블)** — Galaxy / Apple Watch의 활동·수면·심박 리듬 연동. 불규칙 맥박(IRN)은 문진 Q11에 자동 병기됩니다.

### 의료진 웹
- 안압 추세(오늘 실시간 / 기간 추세, 평균+최소·최대 범위), 처방 점안제(제조사·제형·순응도)
- 전자문진 12항목 위험도 요약 — **위험도 배지에 마우스를 올리면 맞춤 안내가 팝업**으로 표시됩니다.
- 점안 순응도 ↔ 안압 **병렬 시각화**(기간 선택), 우선 검토 플래그
- 상관 시각화는 통계적 인과 판정이 아닌 병렬 제시로, SaMD 표시 경계를 준수합니다.

## 실행 방법

```bash
# 1) 의존성 설치
npm install

# 2) 개발 서버 실행
npm run dev

# 3) 프로덕션 빌드 / 미리보기
npm run build
npm run preview
```

브라우저에서 개발 서버 주소(기본 http://localhost:5173)를 엽니다.
상단 토글로 **환자 앱 ↔ 의료진 웹**을 전환할 수 있습니다.

---

## 기술 스택

- **React 18** + **Vite** (빌드/개발 서버)
- **Recharts** (차트: 추세·산점도·상관 병렬 시각화)
- **lucide-react** (아이콘)
- **Tailwind CSS** — 프로토타입 편의를 위해 Play CDN(`index.html`)으로 로드합니다.
  프로덕션 전환 시 PostCSS 기반 Tailwind 빌드로 교체하는 것을 권장합니다.

전체 UI는 단일 컴포넌트(`src/App.jsx`)에 구현되어 있습니다.

## 주의 사항

- 본 저장소는 **프로토타입**이며 화면의 수치·환자 데이터·제품 매핑은 **샘플**입니다.
- 문진 항목·위험도 기준, 측정 신뢰도 정책, 점안–안압 상관 시각화의 표시 범위 등은 **임상 검증 및 규제(SaMD) 검토**가 필요합니다.
- 실제 배포 시 인증/보안, 개인정보·의료데이터 거버넌스(IRB 등), 디바이스 연동(CVT200·스마트 점안·워치) 실연동이 별도로 요구됩니다.
