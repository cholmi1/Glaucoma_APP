# 안압케어 (AnapCare) — 녹내장 통합관리 앱

CVT200 안압계를 중심으로 **안압 · 점안 · 전자문진 · 웨어러블 건강데이터**를 하나의 환자 앱에 통합하고,
임상적으로 의미 있는 데이터를 **의료진 웹**으로 전달하는 녹내장 통합관리 솔루션의 프로토타입입니다.

> C&V Tech · CVT200 Companion · 프로토타입 / 데모용 (샘플 데이터)

## 주요 기능
### 환자 앱
- 홈 — 최근 안압, 점안 순응도 그래프(오늘/2주/1개월/3개월/6개월/1년/누적 + 직접 선택 날짜 지정)
- 안압 — 실시간 다회 측정, 일자별 평균+최소·최대, 좌/우 목표 안압 입력
- 점안 — 약·시간 등록, 30분 전 알림 + 예정 시간 초과 자동 알람, 푸시 알림 토글
- 문진(12항목) — 옵션 선택 + 저장/이전(위험도·안내는 의료진 웹 전용)
- 건강(웨어러블) — Apple/Galaxy Watch 지표 전체. 연속 지표는 오늘/지난밤 + 7일 평균 병기,
  스팟 측정(혈압·체성분·SpO₂ 등)은 측정 시점 표기(2주 경과 시 회색·재측정 권장)
### 의료진 웹
- 안압 추세(실시간/기간+직접 선택), 처방 점안제, 문진 위험도(배지 호버→맞춤 안내), 점안–안압 병렬 시각화(SaMD 경계)

## 푸시 알림
- 활성/백그라운드: Notification API + Service Worker / 완전 종료: 서버 Web Push(VAPID·FCM) 필요
- public/sw.js 수신, src/push.js 구독 스캐폴딩 포함. HTTPS(또는 localhost)에서 동작

## 실행
    npm install
    npm run dev        # http://localhost:5173
    npm run build && npm run preview

- 문진·위험도·측정 신뢰도·상관 시각화 표시 범위는 임상 검증 및 규제(SaMD) 검토 필요.
